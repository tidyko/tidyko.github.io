Charles 设置SSL Proxying抓取https请求

开发调试工具,Charles
  |  
阅读次数
  |  
字数 165
  |  
时长 ≈ 1 分钟

1.

打开Charles 点击 Help -> SSL Proxying -> Install Charles Root Certificate 安装安全证书。

2.

打开Charles 点击 Help -> SSL Proxying -> 安装手机安全证书，根据自己手机系统选择需要安装的选项。

在这里，我使用的是Android手机，所以选择了第二个，点击第二个选项。

3.

点击上一步的选项之后会看到如下界面，会提示你，使用你的手机设置Charles代理，然后打开他给的那个URL就可以下载了。

4.

点击 Proxy -> SSL Proxying Settings ， 设置需要抓取的域名。

5.

上面设置完毕之后就可以抓到指定的https请求了。
